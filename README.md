# ruyi-pytest

## run test

```bash
$ uv run pytest
```

### choose packages-index mirror

Create ``.env``.

iscas mirror <https://mirror.iscas.ac.cn/git/ruyisdk/packages-index/>:

```bash
RUYI_REPO=ISCAS
```

gitee mirror(unofficial) <https://gitee.com/ruyisdk/packages-index/>:

```bash
RUYI_REPO=GITEE
```
